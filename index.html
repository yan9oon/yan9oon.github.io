About Me

학과 중앙대학교 산업보안학과
관심분야 클라우드 보안 아키텍처, 침해사고 대응, 탐지 엔지니어링
주요 경험 AWS 보안 모니터링, Wazuh 룰 튜닝, OCSF 정규화, SOAR 연동

중앙대학교 산업보안학과에서 보안 정책, 관리적 보안, 산업기술 보호, 보안 규정 등을 공부하며 보안이 단순한 기술 문제가 아니라 조직의 자산과 신뢰를 지키는 체계라는 관점을 배웠습니다. 특히 보안 조치가 실제 조직 안에서 어떤 절차와 책임 구조로 작동해야 하는지 고민하며, 정책과 기술을 함께 바라보는 시각을 쌓아왔습니다.

클라우드 보안 프로젝트와 CERT 운영 환경 구축 경험을 통해 로그 수집, 정규화, 탐지 룰 튜닝, 사고 대응 자동화가 어떻게 하나의 흐름으로 연결되는지 직접 경험했습니다. AWS 환경에서 생성되는 로그와 탐지 공백을 분석하고, Wazuh, OCSF, TheHive, Jira 등을 활용해 보안 이벤트가 실제 대응으로 이어지는 구조를 설계하며 실무형 보안 역량을 키워왔습니다.
